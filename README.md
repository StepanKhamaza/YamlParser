# YamlParser

## Описание
Парсинг файлов формата yaml в котлин-объекты

## Используемые технологии
* Kotlin/Jvm - версия 2.1.21
* kotlinx-serialization-json - версия 1.8.1
* charleskorn.kaml - версия 0.78.0
